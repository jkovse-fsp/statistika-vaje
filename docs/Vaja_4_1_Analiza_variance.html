def convert_to_numeric(val):
    if isinstance(val, str) and '-' in val:
        try:
            a, b = map(float, val.split('-'))
            return (a + b) / 2
        except:
            return np.nan
    try:
        return float(val)
    except:
        return np.nan

Glede na pomembne dejavnike za trenerski poklic.

Ker imamo opravka z neodvisnimi kategoričnimi spremenljivkami, uporabimo Kruskal-Wallisov test za primerjavo odzivov na vprašanje Q22. Rezultati so predstavljeni v naslednji tabeli.

q22_cols = ['Q22a', 'Q22b', 'Q22c', 'Q22d', 'Q22e', 'Q22f', 'Q22g', 'Q22h',

results = []

for i, q in enumerate(q22_cols):
    groups = [df_dejavniki_trener[df_dejavniki_trener['Q12'] == group][q].dropna() for group in ['reprezentančni nivo', 'državni nivo', 'ostalo']]

5. Metodologija¶

Preučevali smo razliko med nogometnimi trenerji, ki so aktivno igrali na reprezentančnem, državnem ali drugem nivoju. Gledali smo ali obstajajo razlike med odgovori na vprašanja o pomembnih dejavnikih za trenerski poklic in dejavnikih za razvoj talentov glede na to na katerem nivoju so trenerji igrali, ko so bili še aktivni tekmovalci. Odgovori na vprašanja so bili v obliki Likertove skale (1-5). Za statistično analizo smo uporabili Kruskal-Wallisov test. Meja za statistično značilnost je p<0.05.

Pri pregledu odgovorov glede pomembnosti dejavnikov za trenerski poklic so trenerji različno pomembnost dodelili finančni stabilnosti kluba, možnostim napredovanja, predhodnjim izkušnjam in dosežkom, visokemu dohodku, aktivnem ukvarjanju z nogometom in odnosim z mediji. V ostalih odgovorih ni prišlo do statistično značilnih sprememb.

Pri pregledu odgovorov glede dejavnikov za razvoj talentov so se pokazale statistično značilne razlike pri mnenju glede preprečevanja poškodb in pretreniranosti, zmožnostjo spoprijemanja s pritiski in distanciranja od pričakovanj in vpliva managerjev.
